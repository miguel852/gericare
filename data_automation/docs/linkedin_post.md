# Post para LinkedIn

Desenvolvi um projeto de automacao de dados para o GeriCare, uma solucao voltada para
cuidado geriatrico.

O objetivo foi transformar planilhas operacionais em bases confiaveis para analise,
acompanhamento de risco e apoio ao plantao.

Stack usada:

- Python
- Apache Spark / PySpark
- Docker
- Databricks
- Arquitetura bronze, silver e gold

O pipeline:

- Le planilhas de residentes, sinais vitais, medicacoes e tarefas.
- Padroniza campos, tipos, nomes, contatos e datas.
- Remove duplicidades e aplica regras de qualidade.
- Calcula score de risco por residente.
- Gera datasets finais para quadro de plantao e follow-up familiar.

Esse projeto reforcou conceitos de engenharia de dados aplicados a um contexto real de
saude: qualidade de dados, transformacao distribuida, rastreabilidade e entrega de
informacao operacional.
